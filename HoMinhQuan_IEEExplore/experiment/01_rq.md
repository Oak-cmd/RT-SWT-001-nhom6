# Research Question (RQ)

**Study:** GPT-4o Zero-Shot Gherkin Generation from Connextra User Stories  
**SLR basis:** 16 included papers · Search date: May 27 & 31, 2026 · 122 records after deduplication

---

## Initial RQ (before SLR)

> "Can an LLM automatically generate Gherkin acceptance test scenarios from user stories that are semantically similar to expert-written scenarios and syntactically executable?"

---

## PICO (initial)

| Element | Description |
|---------|-------------|
| **P** (Population) | Software projects with user stories as requirements |
| **I** (Intervention) | LLM-based zero-shot generation of Gherkin acceptance test scenarios |
| **C** (Comparison) | Expert-written Gherkin scenarios for the same user stories |
| **O** (Outcome) | Semantic similarity and syntactic executability of the generated scenarios |

## Refined RQ (final)

> "Does GPT-4o zero-shot (temperature = 0) generate Gherkin acceptance test scenarios that achieve cosine semantic similarity ≥ 0.85 with expert-written scenarios AND executable syntax rate ≥ 80% when applied to Connextra-format user stories from SE projects?"

---

## PICO (refined)

| Element | Description |
|---------|-------------|
| **P** (Population) | Connextra-format user stories from ≥ 3 independent SE projects |
| **I** (Intervention) | GPT-4o zero-shot prompting (temperature = 0) to generate Gherkin scenarios + step definitions |
| **C** (Comparison) | Gherkin scenarios written by an expert software tester for the same user stories |
| **O** (Outcome) | **Metric 1:** cosine semantic similarity (all-MiniLM-L6-v2) ≥ 0.85 · **Metric 2:** Cucumber-executable syntax rate ≥ 80% |

---

## Justification of Thresholds

### Similarity threshold (≥ 0.85)
- Alinezhadtilaki & Evans (2025) — the only paper to measure cosine similarity — report mean BERT-cosine = 0.79 for scenarios rated "acceptable". We adopt 0.85 as a stricter, research-grade bar.
- The all-MiniLM-L6-v2 encoder operates on a sentence-level training objective and produces higher absolute cosine values than BERT token embeddings, making 0.85 a meaningful and comparable threshold.
- No prior paper has established or validated this threshold — confirming it as a new empirical contribution.

### Executable syntax threshold (≥ 80%)
- Few-shot upper range: Fonseca et al. (2025) 83%, Ferreira et al. (2025) 81%, Cypress Copilot (2025) 87%.
- Zero-shot baseline: Karpurapu et al. (2024) GPT-4 zero-shot achieves 76% syntactic validity.
- 80% is set as the minimum acceptable threshold for zero-shot output — acknowledging that zero-shot is expected to underperform few-shot, but must still clear the practical usability bar.
