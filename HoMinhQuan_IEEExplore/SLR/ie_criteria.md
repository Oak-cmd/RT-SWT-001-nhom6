# Inclusion / Exclusion Criteria

**Topic:** LLM for Acceptance Test Automation (BDD/Gherkin)  
**Date:** May 2026

---

## Inclusion Criteria (IC)

A paper must satisfy **all** of the following criteria to be included.

| ID  | Criterion |
|-----|-----------|
| IC1 | Written in English |
| IC2 | Published in 2018 or later |
| IC3 | Peer-reviewed (journal or refereed conference proceedings) |
| IC4 | Related to at least one of: automated test generation, Gherkin/BDD, or LLM applications in Software Engineering |
| IC5 | Reports empirical results with quantitative data (surveys and opinion papers are excluded) |

---

## Exclusion Criteria (EC)

A paper is excluded if it meets **any** of the following criteria.

| ID  | Criterion |
|-----|-----------|
| EC1 | Duplicate of a paper already included |
| EC2 | Full text (PDF) not accessible |
| EC3 | Poster, extended abstract, or fewer than 4 pages |
| EC4 | Focuses solely on test execution or test maintenance — does not address test generation |
