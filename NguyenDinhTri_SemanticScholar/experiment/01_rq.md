1. Initial Research Question (RQ)
"For user stories written in Connextra format (P), can an LLM automatically generate Gherkin scenarios and step definitions (I) compared to manually written BDD scenarios (C) to achieve a semantic similarity ≥ 0.85 with expected behavior and executable syntax without errors (O)?" 1
2. Initial PICO
Population (P): User stories in Connextra format ("As a... I want... So that..."). 1
Intervention (I): LLM automatically generating Gherkin scenarios and step definitions. 1
Comparison (C): BDD scenarios manually written by experts. 1
Outcome (O): Semantic similarity ≥ 0.85 and executable syntax (no parsing errors). 1
